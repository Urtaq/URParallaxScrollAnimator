## What is this?
Show an animation as far as moved scroll while scrolling at the scroll view

## Requirements

* iOS 8.1+
* Swift 3.0+

## Installation

### Cocoapods

Add the following to your `Podfile`.

    pod "URScrollAnimator"

## Examples

See the `Example` folder.  
Run `pod install` and open the .xcworkspace file.
